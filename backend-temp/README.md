# backend-temp (legacy / غیرفعال)

این پوشه یک **اسکافلد قدیمی Laravel** (با Vite) است و بخشی از برنامهٔ اصلی **Webino** نیست.

برنامهٔ فعال در مسیر [`../backend/`](../backend/) است (Laravel 11 + `nwidart/laravel-modules`).

- برای توسعهٔ API و دیتابیس از `backend/` استفاده کنید.
- اگر این پوشه برای شما سردرگمی ایجاد می‌کند، می‌توان آن را در آینده از repo حذف کرد؛ تا آن زمان این فایل به‌عنوان برچسب «deprecated» کافی است.

## ویژگی‌های فقط وردپرس (webinocrm)

موارد زیر در پلاگین مرجع `webinocrm` وابسته به **وردپرس / ووکامرس** هستند و در استک Laravel+Next به‌صورت پیش‌فرض معادل یک‌به‌یک ندارند:

- یکپارچگی Elementor، شورتکدها و CPT وردپرس
- Cron و ایمیل و رسانهٔ وردپرس
- پنل ادمین WooCommerce برای بستهٔ Bale Business

برای مهاجرت رفتار، یکپارچه‌سازی جداگانه یا ماژول Laravel طراحی شود.
